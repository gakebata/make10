# make10

入力した数字を四則演算 (`+ - * /` と括弧) で組み合わせて、**目標値 (既定 10) になる式を列挙する**パズルソルバーです。

## ブラウザ版 (インストール不要)

**<https://gakebata.github.io/make10/>**

スマートフォン (iPhone 等) でもそのまま動作します。実体は依存ゼロの単一ファイル [docs/index.html](docs/index.html) で、コアロジックを JavaScript に移植し `BigInt` ベースの厳密分数で計算するため、Python 版と同様に浮動小数点誤差による取りこぼしはありません。

**PWA 対応** — Safari (iPhone) や Chrome で開き、共有メニューから「ホーム画面に追加」すると、アドレスバーのない全画面アプリとして起動できます。Service Worker がアプリ一式をキャッシュするため、一度読み込めば**以後はオフラインでも動作**します。

## 特長

- **厳密計算**: [`fractions.Fraction`](https://docs.python.org/ja/3/library/fractions.html) を使い、`8 / (3 - 8 / 3) = 24` のような割り算を含む解も浮動小数点誤差なく発見します。
- **`eval` 不使用**: 式を木構造として組み立てて評価するため安全です。
- **任意個数に対応**: 数値の個数に依らず、全ての順序・グループ化・演算子の組み合わせを再帰的に網羅します。
- **重複除去**: 可換則・結合則を考慮した正規化により、`1 + 2` と `2 + 1` のような等価な式をまとめて表示します。
- **括弧最小化**: 演算子の優先順位に従い、必要最小限の括弧で見やすく表示します。

## インストール

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"   # 開発用 (pytest, ruff 込み)。利用だけなら ".[dev]" を外す
```

## 使い方

### コマンドライン

```sh
make10 1234               # 1, 2, 3, 4 で 10 を作る
make10 --target 24 1234   # 目標値を 24 に
make10 "12 3 4"           # 空白/カンマ区切りで複数桁の数値
make10 --no-dedup 1234    # 重複を除去せず全て表示
make10                    # 引数なしで対話モード
```

実行例:

```text
$ make10 1234
1 + 2 + 3 + 4 = 10
4 * (3 - 1 / 2) = 10
2 + 4 * (3 - 1) = 10
...
[Count: 30]
```

### ライブラリとして

```python
from make10 import find_solutions

for formula in find_solutions("1234", target=10):
    print(formula)
```

## 入力形式

- 区切りなし (`"1234"`): 旧実装と同様に **1 文字ずつ** の数値として扱います。
- 空白・カンマ区切り (`"12 3 4"`, `"12,34"`): **複数桁** の数値として扱います。
